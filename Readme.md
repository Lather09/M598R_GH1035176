## 📌 Project Overview

This project presents an **Exploratory Data Analysis (EDA)** of the Coveo Fashion E-Commerce dataset, originally released as part of the **Shopper Intent Prediction challenge** (Requena et al., 2020).

The goal of this notebook is to:
- Understand user behavior in an e-commerce setting
- Explore patterns in browsing and purchasing activity
- Prepare the dataset for future predictive modeling tasks (e.g., purchase prediction, recommendation systems)

---

## 📂 Dataset Information

- **Dataset Name:** Coveo Shopper Intent Dataset  
- **File Used:** `release_10_23_2020.csv`  
- **Domain:** Fashion E-Commerce  
- **Data Type:** User sessions, product interactions, and behavioral signals

## 🔍 Key Tasks Performed

### 2. Data Cleaning

### 3. Exploratory Data Analysis (EDA)

- Product interaction patterns

- Initial insights for downstream ML tasks

## 📊 Insights & Observations

- Users exhibit distinct browsing vs purchasing behaviors
- Certain product categories drive higher engagement
- Session patterns indicate potential for **intent prediction modeling**
- Data is suitable for:
  - Classification (purchase prediction)
  - Recommendation systems
  - Sequence modeling
